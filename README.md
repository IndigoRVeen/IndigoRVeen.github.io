# Indigo Veen GitHub Pages

Deze repository bevat een mobiele homepage voor Indigo Veen, gemaakt als GitHub Pages landing page.

## Inhoud

- `index.html` – nieuwe homepage met focus op diensten, contact en branding
- `README.md` – dit overzicht
- `jarvis-dashboard.html`, `largo-gok-junk-1.html`, `serve.ps1` – overige voorbeelden en lokale serverbestanden

## Installatie

1. `git init`
2. `git add .`
3. `git commit -m "Initial commit: add GitHub Pages homepage for Indigo Veen"`
4. Voeg je GitHub remote toe:
   `git remote add origin https://github.com/<gebruikersnaam>/<repo>.git`
5. `git push -u origin main`

## GitHub Pages

Plaats `index.html` in de root van de repository of in de `docs/`-map en schakel GitHub Pages in via de repository-instellingen.
